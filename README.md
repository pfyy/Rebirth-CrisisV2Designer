# 重生之我是危机合约设计师

## 如何同时按死小刻和所有物理伤害

愚蠢的做法：敌方防御+10000000%

正确的做法：敌方获得琉璃壁，琉璃壁能够抵挡物理伤害，琉璃壁的防御力+10000000

## 如何设计控制卡

1. 设计XX控制效果，让现有的敌人没有XX抗性

2. 设计危机合约，让有XX控制效果的角色YY登顶

3. 全面普及XX抗性，让角色YY爆金币

4. 回到步骤1

## 如何设计日常Boss

有什么抗性就加什么抗性，物法减伤99%，物法闪避99%，最好是超高dph的5连击远程打10物法混伤（dps当然也不能落下，攻速也要快）+九宫格溅射
